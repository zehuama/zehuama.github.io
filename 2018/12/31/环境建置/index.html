环境建置

安装 Xcode

直接在app store中下载

确认安装 Command Line Tools

在terminal中输入：
xcode-select --install

如何确定已经装好
在Terminal视窗输入：
xcode-select -p

显示出： /Applications/Xcode.app/Contents/Developer
即是成功安装！

确认安装 Homebrew

打开 Homebrew 官方网站：http://brew.sh/index_zh-tw.html
复制里面那行代码，在terminal跑

发现 homebrew 需要使用先同意 Xcode 的使用者规范

输入：sudo xcodebuild -license 同意。

  • 中间会出现很多条文，一直按“空白键”卷到最下页，然后输入 agree 同意。

在 Homebrew 安装 git

在 Terminal 视窗输入：
brew install git

安装 ImageMagick

brew install imagemagick

安装 PostgresSQL

brew install postgresql

安装完毕后：

brew services start postgresql （会确保在你下次开机时，也一并启动资料库）

确认安装 RVM

RVM 的官方网址是 http://rvm.io

官方网站提供的安装指令是：

\curl -L https://get.rvm.io | bash -s stable

安装完输入：source ~/.rvm/scripts/rvm 让 rvm 生效。

然后安装一个套件

brew install libxml2

确认安装 Ruby

rvm install 2.3.1 <== 尾端可选择自己需要的版本 EX: 1.9.3 , 2.1.2 … etc

rvm use 2.3.1 --default 使用 ruby 2.3.1 为预设版本 (可改自己需要的版本)

安装完以后记得重开 terminal。

rvm list 列出目前已安装的 Ruby 版本。

指定 RubyGems 来源（参考，如果你人在内地，无法连上再使用这个方法）
如果遇到连不太上 https://rubygems.org。
可以多执行一行：

gem sources --add https://ruby.taobao.org/ --remove https://rubygems.org/

确认安装 Rails

（建议你在连上 vpn 的环境上操作。因为墙是挡掉 rubygems.org 的）

gem install rails -v 5.0.0 (后面的参数是指定版本，预设会抓最新版)

Atom / Sublime

在编程的时候，除了输入指令之外，我们还需要能够编辑程序的软件。

目前市面上有许多各种不同的 Rails 编辑器，
这里我们推荐给新手的有两种，可以选择一种安装就可以了：Atom / Sublime

Atom

一套是由 Github 维护的 Atom http://atom.io

下载后，解压缩，将 atom 拖拉至 “应用程式”即可。

Sublime

一套是独立开发者所维护的 Sublime https://www.sublimetext.com/

下载后，双击 Submlime 打开 dmg, 将 Sublime 拖到应用程式即可

设定在 Terminal 内可以打开 Atom 与 Sublime

如何在 Terminal 内可以“用惯用的编辑器”快速打开“目前正在做”的专案（project）呢？

这个功能叫 Shell commands。这个功能可以帮助你在最短的时间内，开启正在执行的专案程序码档案。

Atom 内开启这个功能的选项，在左上角的下拉选单里：

  • Atom 版本

  • Sublime 版
    sudo ln -s "/Applications/Sublime Text.app/Contents/SharedSupport/bin/subl" /usr/local/bin/subl
    （会需要输入这台Mac的密码）

未来当我们在制作一个新的专案（project）的时候，就可以透过在Terminal输入指令，让Atom或Sublime自动打开相对应的资料夹。
