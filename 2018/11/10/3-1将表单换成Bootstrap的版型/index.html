Simple Form 表單產生器

SimpleForm 是一套強大的表單產生器，它能自動產生 Bootstrap 的表單版型。在這裡我們將會使用這套 gem 換掉我們未來的所有表單產生方式。

安裝 simple_form

執行 bundle install

安裝 simple_form for bootstrap 的設定

rails generate simple_form:install --bootstrap

重開 rails server

（注意：裝任何 Gem 都要重開 rails s）

重開 rails s
