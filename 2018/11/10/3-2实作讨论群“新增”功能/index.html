3.2 实作讨论群“新增”功能

目标

这一章我们要实作 讨论群 的新增功能

步骤

Step 0. git checkout -b ch03

因为我们现在要实作第三章的内容，新增一个 branch 把实作内容集中在 ch03 是比较合理的。将来回顾就可以比较清楚 在 ch03 我们做了哪一些练习。

git checkout -b ch03

Step 1 : 新增看板

建立 “new” action

建立 “new” 的表格 template

touch app/views/groups/new.html.erb

git 存档

git add .
git commit -m "implement groups#new "

前往 http://localhost:3000/groups/new 即會出現新增討論版的表單




目前還只是個表單畫面，無法將資料送出後成功建立出一筆討論版的資料，缺了 action ‘create’

Step 2 : 接收資料並儲存

建立 “create” action

我們要將 http://localhost:3000/groups/new 裡的表單送出的訊息變成一筆存在資料庫裡的資料

完成後再送一次即可成功建立資料
可以多建幾筆資料測試功能

git 存档

git add .
git commit -m "implement groups#create"

group_params 又是什么？

这是一个“过滤器架构”叫 Strong Parameter，在这里你先不用详细理解 Strong Parameter 是什么，你只要知道因为 Rails 很聪明的会翻译你的表格，所以外面的坏人会想要偷塞一些垃圾进来，所以我们要设一个“允许进来的栏位”，在这里就是title与description
