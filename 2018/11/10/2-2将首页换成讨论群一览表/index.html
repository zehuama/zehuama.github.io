2.2 将首页换成讨论群一览表

目标

在这一节，我们会完成以下事项：

  • 使用者在首页，可以直接看到“讨论群一览表”

    步骤

    Step 1. 修改 routing

    把 root 'welcome#index' 删掉，换成 root 'groups#index'

（记得 + - 不要打进去，他们各代表“加这一行” / “减这一行”的意思。）

然后你就可以在 http://localhost:3000/ 看到 Group 一览表了

Step 2. git 存档

git add .
git commit -m "replace root with groups#index"
