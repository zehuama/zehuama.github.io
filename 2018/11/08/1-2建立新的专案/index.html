1.2 建立新的专案


目标

  • 这一节的目标，是要带领大家建立一个新的全新 Rails 空专案。

步骤

Step 1. 确认 Ruby 版本

输入 ruby -v
确认一下现在的 Ruby 版本。

Step 2. 确认 Rails 版本（版本 5.0.0 或以上)

输入 rails -v
确认现在的Rails版本 。

Step 3. 建立一个新 Rails 专案

输入 rails new rails101 (我新建的folder是rails101-7)

Step 4. 进入新建好的专案, 并建立 git 做版本控制

请跟着指南，手打输入每一行指令。

cd rails101
git init
git add .
git commit -m "Initial Commit"

解说

未来在做这个专案练习时，记得每做一个章节，就把程式码 commit 一次。等于每一个章节都建立一个存档点。
这样子的话，如果之后改烂，或是改出来你不喜欢，你都可以利用 git 来“还原”到先前的状态。
